# Junior Full Stack Developer Portfolio

A comprehensive portfolio website showcasing junior-level full stack development skills, built with modern web technologies.

## 🚀 Project Overview

This portfolio demonstrates understanding of full stack web development through:
- **Frontend Technologies**: HTML5, CSS3, Bootstrap Framework, JavaScript (ES6+)
- **Backend Concepts**: PHP server-side development, API design, database interactions
- **Full Stack Integration**: How frontend and backend technologies work together

## 📁 Project Structure

```
junior-dev-portfolio/
├── index.html                 # Homepage with portfolio overview
├── fullstack-development.html # Full stack development concepts & PHP
├── html.html                  # HTML fundamentals & semantic markup
├── css.html                   # CSS styling, layouts & responsive design
├── bootstrap.html             # Bootstrap framework components & utilities
├── javascript.html            # JavaScript interactivity & DOM manipulation

```
