Understanding JavaScript

The programming language that brings web pages to life with interactivity

What is JavaScript?

JavaScript is a programming language that runs in web browsers to make web pages interactive. It allows you to respond to user actions (clicks, typing, scrolling), modify content, validate forms, perform calculations, and communicate with servers.

Without JavaScript, websites would be static - unable to respond to user input or change dynamically. JavaScript is essential for modern web development.

JavaScript Basics

These core concepts form the foundation of JavaScript. Understanding them makes it easier to read, write, and debug code.

  • Variables: Store and update values. Use const for constants and let when the value can change.
  • Data Types: Includes strings, numbers, booleans, arrays, and objects; each type behaves differently in operations.
  • Functions: Wrap reusable logic into named actions. Functions can accept inputs (parameters) and return results.
  • Events: Let code respond to user actions like clicks, typing, and form submission.
  • DOM Manipulation: JavaScript can read and update HTML elements (text, attributes, styles) to change the page dynamically.
  • Control Flow: Use if/else and loops (for, while) to make decisions and repeat actions.

Common JavaScript Concepts

What this does: Declares variables of several data types and uses typeof to show the type of a value.

Functions

Functions are reusable blocks of code. They can accept inputs (parameters) and return values.

// Function declaration
function greet(name) {
return `Hello, ${name}!`;
}

// Arrow function (shorter syntax)
const add = (a, b) => a + b;

let message = greet("Aayush");
console.log(message); // "Hello, Aayush!" console.log(add(2, 3)); // 5

What this does: Defines reusable functions, calls them with arguments, and logs the results. The arrow function is a shorter way to write simple functions.

Event Listeners & DOM Interaction

Event listeners run code when users interact with the page, and the DOM is how JavaScript reads/changes HTML.

// Respond to a button click and update the page
const button = document.querySelector("#myButton");
const output = document.querySelector("#output");

button.addEventListener("click", () => {
output.textContent = "Button clicked!"; // Update DOM content
});

What this does: Selects elements from the page, listens for a click event, and changes the text on the page when the event happens.

Tip: Use console.log() to debug values and check what your code is doing.
